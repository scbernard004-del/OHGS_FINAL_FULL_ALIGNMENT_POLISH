OHGS FULL WEBSITE - READY FOR GITHUB + VERCEL

WHAT IS INSIDE
- All website pages
- style.css
- script.js
- assets folder
- images folder
- videos folder
- favicon.png
- all current fixes and alignment updates

IMPORTANT
Upload everything inside this folder/ZIP to the SAME GitHub repository.
Do not create a new repository.
Do not delete your domain in Vercel.
Your domain stays connected as long as Vercel project stays the same.

GITHUB UPLOAD STEPS - COMPUTER / OPERA BROWSER
1. Download this ZIP.
2. Extract/unzip it on your computer.
3. Open GitHub.
4. Open your current repository.
5. Click Add file.
6. Click Upload files.
7. Open the extracted folder.
8. Select ALL files and folders inside it:
   - all .html files
   - style.css
   - script.js
   - favicon.png
   - assets folder
   - images folder
   - videos folder
   - README files
9. Drag them into GitHub upload area.
10. GitHub will say some files already exist. That is okay.
11. Scroll down.
12. Choose Commit directly to the main branch.
13. Click Commit changes.

GITHUB UPLOAD STEPS - SAMSUNG PHONE
1. Download this ZIP.
2. Open My Files on Samsung.
3. Go to Downloads.
4. Tap the ZIP.
5. Tap Extract.
6. Open GitHub in Opera.
7. Open your repository.
8. Tap Add file then Upload files.
9. Tap choose your files.
10. Select the extracted website files.
11. If your phone allows folder upload, select:
    - assets
    - images
    - videos
    - all .html files
    - style.css
    - script.js
    - favicon.png
12. If your phone does not allow folder upload, use a computer. GitHub on phone is weak with folders.
13. Scroll down.
14. Select Commit directly to main.
15. Tap Commit changes.

VERCEL STEPS AFTER GITHUB
1. Open Vercel.
2. Open the same OHGS project.
3. Go to Deployments.
4. Wait for the newest deployment to appear.
5. It should start automatically after GitHub commit.
6. When it says Ready, open your domain:
   https://ohgs.co.tz
7. If it still shows old website, refresh the page.
8. On phone, open the site, tap browser menu, then reload/refresh.
9. If needed, open in private tab once to check.

DO NOT CHANGE DOMAIN SETTINGS
- Do not remove ohgs.co.tz from Vercel.
- Do not delete DNS records in Sakura.
- Do not make a new Vercel project.
- Only update GitHub files. Vercel will redeploy automatically.

IF VERCEL DOES NOT UPDATE
1. Open Vercel.
2. Open project.
3. Click Deployments.
4. Click the newest deployment.
5. If it failed, open Logs.
6. If it is Ready but the website still looks old, click Redeploy from the three dots on the latest deployment.
7. Wait until Ready again.

LOCAL PREVIEW
1. Extract the ZIP.
2. Open index.html in your browser.
3. Check pages before uploading.
4. Some video or browser features may work better after upload to Vercel.
